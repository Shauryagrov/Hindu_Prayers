# Quick Test Checklist

Use this after each step. Check off items as you test.

## Pre-Testing
- [ ] Run `./quick_test.sh` or build in Xcode (`Cmd + B`)
- [ ] Build succeeds with no errors

## Core Functionality (5 min)
- [ ] App launches
- [ ] Welcome screen works
- [ ] All 4 tabs accessible
- [ ] Can navigate between tabs

## Hanuman Chalisa (5 min)
- [ ] Verses tab loads
- [ ] Can open verse detail
- [ ] Hindi audio plays
- [ ] English audio plays
- [ ] Audio works in silent mode
- [ ] Complete tab works
- [ ] Quiz tab works

## New Feature (Test what you just added)
- [ ] New feature works
- [ ] No crashes related to new feature
- [ ] Integrates well with existing features

## Regression (5 min)
- [ ] Existing features still work
- [ ] No data loss (bookmarks, progress)
- [ ] Performance is acceptable

## If Everything Passes
- [ ] Commit changes
- [ ] Ready for next step

## If Issues Found
- [ ] Document issue
- [ ] Fix before proceeding
- [ ] Re-test

---

**Total Time: ~15 minutes per step**
